# COVID-19 Dashboard

This project analyzes COVID-19 data using Python, pandas, and matplotlib.

##  Technologies Used
- Python
- pandas
- matplotlib

## Features
- COVID-19 Cases Analysis
- Death Analysis
- Recovery Analysis
- Trend Visualization

##  Charts Used
- Line Chart
- Trend Graph

##  Dataset Columns
- Day
- Cases
- Deaths
- Recoveries

## How to Run

Install required libraries:

```bash
pip install pandas matplotlib
```

Run the project:

```bash
python app.py
```

## Output
- Daily cases graph
- Recovery trend graph
- COVID-19 analysis dashboard

##  Project Goal
To analyze pandemic trends and visualize COVID-19 data using data analysis and visualization techniques.
